# SAO Event Management Platform

## Overview
The **SAO Event Management Platform** is designed to improve event management efficiency at **Al Akhawayn University** and enhance the student experience. The platform simplifies planning, execution, and participation in university events, providing a seamless and intuitive interface for both students and administrators.

---

## Key Features
- **Event Management:** Create, modify, and delete events with ease.
- **Ticketing System:** Generates automated, one-time QR-coded tickets.
- **Student Dashboard:** Interactive interface to view, filter, and manage bookings.
- **History & Analytics:** Track attendance and engagement for students and admins.
- **Google Calendar Integration:** Sync events with students’ personal calendars and receive email notifications.

---

## Stakeholders
- **Students, Staff, and Faculty:** Main users for booking and attending events.
- **SAO Staff & Managers:** Responsible for event creation and administration.
- **IT Department:** Provides technical support and maintenance.
- **Human Resources / Developers:** Feedback, development, and testing support.

---

## Technologies
- **Front-End:** HTML, CSS, React.js
- **Back-End:** Java
- **Database:** To be confirmed (university database or new)
- **APIs:**  
  - Google Calendar API – for event notifications  
  - QR Code API (GoQR) – for ticket generation

---

## How It Works
1. **Admins** create and manage events through the platform.
2. **Students** browse events, book tickets, and sync events with Google Calendar.
3. **QR-coded tickets** are generated automatically for each booking.
4. Attendance and engagement history are tracked for analytics.
